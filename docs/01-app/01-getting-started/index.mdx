Boas vindas à documentação do Next.js!

Esta seção **Introdução** ajudará você a criar sua primeira aplicação Next.js e
a aprender os principais recursos que você usará em todos os seus projetos.

## Conhecimentos necessários

Nossa documentação pressupõe alguma familiaridade com desenvolvimento web.
Antes de começar, é recomendável que você tenha familiaridade com:

- HTML
- CSS
- JavaScript
- React

Se você é iniciante em React ou precisa relembrar os conceitos básicos,
recomendamos começar com o nosso
[curso React Foundations](/learn/react-foundations) e o
[curso Next.js Foundations](/learn/dashboard-app), que permite que você construa
uma aplicação enquanto aprende.
