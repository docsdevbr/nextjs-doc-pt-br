> Consulte a documentação do [App Router](/docs/app) para obter informações
> sobre os recursos mais recentes.

O **Pages Router** usa um roteador de sistema de arquivos intuitivo para mapear
cada arquivo a uma rota.

Antes do Next.js 13, o Pages Router era a principal maneira de criar rotas no
Next.js.
Ele ainda é compatível com versões mais recentes do Next.js, mas recomendamos a
migração para o novo [App Router](/docs/app) para aproveitar os recursos mais
recentes do React.
