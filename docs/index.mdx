Boas vindas à documentação do Next.js!

## O que é o Next.js?

Next.js é um framework React para construir aplicações web full-stack.
Você usa componentes React para construir interfaces da pessoa usuária e o
Next.js para recursos e otimizações adicionais.

Ele também configura automaticamente ferramentas de baixo nível, como bundlers e
compiladores.
Assim, você pode se concentrar em construir seu produto e lançá-lo rapidamente.

Seja você uma pessoa desenvolvedora individual ou parte de um time maior, o
Next.js pode te ajudar a criar aplicações React interativas, dinâmicas e
rápidas.

## App Router e Pages Router

O Next.js possui dois roteadores diferentes:

- **App Router**: O roteador mais recente que oferece suporte a novos recursos
  do React, como Componentes de Servidor.
- **Pages Router**: O roteador original, ainda com suporte e em constante
  aprimoramento.

Na parte superior da barra lateral, você notará um menu suspenso que permite
alternar entre a documentação do [App Router](/docs/app) e do
[Pages Router](/docs/pages).

### Gerenciamento de versões do React

O App Router e o Pages Router lidam com as versões do React de maneiras
diferentes:

- **App Router**: Usa as versões canary do React
  (https://react.dev/blog/2023/05/03/react-canaries), que incluem todas as
  alterações estáveis do React 19, bem como novos recursos que estão sendo
  validados em frameworks, antes do lançamento de uma nova versão do React.

- **Pages Router**: Usa a versão do React instalada no arquivo `package.json` do
  seu projeto.

Essa abordagem garante que os novos recursos do React funcionem de forma
confiável no App Router, mantendo a compatibilidade com versões anteriores para
aplicações existentes do Pages Router.

## Conhecimento prévio necessário

Nossa documentação pressupõe alguma familiaridade com desenvolvimento web.
Antes de começar, é importante que você tenha familiaridade com:

- HTML
- CSS
- JavaScript
- React

Se você é iniciante em React ou precisa relembrar os conceitos básicos,
recomendamos começar com o nosso
[curso de Fundamentos de React](/learn/react-foundations) e o
[curso de Fundamentos de Next.js](/learn/dashboard-app), que permite criar uma
aplicação enquanto aprende.
